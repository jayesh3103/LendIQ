# LendIQ

LendIQ is a web application designed to assist users in managing their financial loans with the help of artificial intelligence. The application provides a user-friendly interface for calculating loans, tracking payments, and accessing financial insights.

## Features

- **Loan Calculator**: Easily calculate loan amounts, interest rates, and repayment schedules.
- **Dashboard**: View your financial overview and track your loan status.
- **User-Friendly Interface**: Intuitive design with a lavender color scheme for a pleasant user experience.
- **Localization**: Supports English (India) for a tailored experience.

## Installation

To get started with LendIQ, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd LendIQ
npm install
```

## Usage

To run the application in development mode, use the following command:

```bash
npm start
```

This will start the application on `http://localhost:3000`.
